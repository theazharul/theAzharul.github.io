Convert org heading to Todo item => S \<Right Arrow\>  
Add new tag => SPC m q  
Schedule a todo => SPC m d s  
Set Deadline for todo => SPC m d d  
Org Agenda view => SPC o a a a

Key description:  
Space => SPC  
Shift => S
